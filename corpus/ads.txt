·       UvA has to hold N exams in one day and can take place in any room 
·       Each exam is scheduled from a starting time to an end time 
·       Use graph coloring and assign each node a color where adjacent nodes have different colors 
·       For the scheduling exam the time complexity is N^2-n/2 so O(N^2)
·       For nested loops like for i in (1, N) and the for j in (i, N) the complexity becomes N(N-1)/2
·       Recursive coloring: solution for the graph = safely color a node + solution for the graph without this node
·       The pseudo code:
-              Start with N=0, so you color the first node
-              If N not in G return OK, this is because you have successfully colored all nodes and went beyond number of nodes
-              If N is colored return process (N+1), colors N then moves on to the next node
-              For each safe color C in K: iterate through all available colors K and check ig it can be safely assigned to current node N and doesn’t conflict with colors of adjacent nodes
-              Return no solution: if no safe coloring can be found for node N and must backtrack to a different color
·       Best case: after coloring a node, subproblem is solved without backtracking = process called once per node; complexity = O(V+E)
·       Worst case: subproblems are all solved by backtracking from last node = process (N+1) called Safe(N) times for node N; O(phi**V)
